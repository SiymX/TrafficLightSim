# TrafficLightSim

TrafficLightSim is a simple four way traffic light and traffic flow simulator built with HTML, CSS, and JavaScript.

Created by **Khandakar Sayeem** and **Khandakar Mahin**.

---

## Live demo

Test the live demo here:  
https://siymx.github.io/TrafficLightSim/

---

## Features

- Four way intersection rendered on an HTML canvas  
- Multiple lanes in each direction  
- Automatic signal cycle for north–south and east–west  
- Manual mode to force NS green, EW green, or all red  
- Adjustable traffic density slider  
- Basic car following, braking, and lane merging  
- Live stats for:
  - Current phase  
  - Cars on road  
  - Cars passed  
  - Average wait time at red
